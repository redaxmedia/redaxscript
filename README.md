Installation
------------

Clone the repository:

```
git clone https://github.com/redaxscript/redaxscript.git --branch=4.4.0
```

Install the dependencies:

```
composer install
```

```
npm install
```


Usage
-----

Run local development server:

```
grunt serve [options]

-L, --live-reload
-O, --open-browser
-N, --no-cache
-D, --debug-mode
```

Browse your working copy:

```
http://localhost:8000
```
